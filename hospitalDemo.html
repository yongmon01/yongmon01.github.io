Hospital Management Demo

초기 페이지의 모습입니다. 환자의 리스트가 있고 AllowPaging 으로 페이지를 나누었습니다.

데이터 정렬

각 컬럼의 이름을 클릭하면 정렬이 가능합니다. 환자의 성을 기준으로 정렬 해보았습니다.

데이터 조회

환자를 선택하면 해당 환자의 방문기록을 열람 합니다. 강호동 환자를 선택해 보았습니다. 강호동 환자의 방문기록 테이블이 나타납니다. 강호동 환자의 누적 진료비도 페이지에 나타내고 있습니다
강호동 환자의 2020년 5월 3일 방문을 선택해 보았습니다. 해당 방문에 대한 처방정보가 담긴 테이블이 나옵니다.

데이터 업데이트

환자의 이름 또는 주소는 편집이 가능합니다. 강호동 환자의 주소를 송파구로 업데이트 해보겠습니다.
강호동 환자의 주소가 잘 변경되었습니다.

데이터 삽입

처음 방문한 유아인 환자를 입력해 보겠습니다.
유아인 환자가 잘 입력되었습니다. 방문기록과 처방기록 까지 입력해 보겠습니다.
성공적으로 입력이 되었습니다.

데이터 삭제

유아인 환자의 방문기록 삭제를 시도하였지만 해당 방문에대한 처방전이 있기 때문에 삭제할 수 없다는 알림을 표시합니다.
처방전을 삭제후에 방문기록을 삭제 해보았습니다. 성공적으로 적용이 되는 것을 볼 수 있습니다.
